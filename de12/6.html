第6回　プログラミング

  • 今回は画像の画質を調べ、平均画質で並び替えるプログラムを作ろうと思った。
  • コード
    1. from PIL import Image
    2. import numpy as np
    3. import matplotlib.pyplot as plt
    4. FILE_PATH = 'けん玉.jpg'
    6. # 画像ファイルパスから読み込み
    7. img = Image.open(FILE_PATH)
    8. img = img.convert("RGB")
    10. # 画像の表示
    11. plt.imshow(img)
    12. plt.show()
    13. # データの確認
    14. print("データ：", img)
    15. img_array = np.asarray(img)
    16. print("配列：", img_array)
    18. #画素数を出す
    19. print(img_array.shape) #(縦の画素数,横の画素数,RGB)
    20. #numpy.meanで平均を求める
    21. import numpy
    23. list1 = [img_array.shape]
    24. print(list1)
    25. print('・numpy.average()で平均値を求める')
    26. list1_ave = numpy.mean(list1)
    27. print('平均値は', list1_ave)

  • やっぱりプログラミングは難しい。思っているように動かすコードの組み合わせを
    探すのが大変だった。このコードでは実際にやりたい動きができなかった。
    平均の画素もうまく求められなかったし、複数の画像を一気に処理し並び替えることもできなかった。
